# NexusAPI - Decentralized API Marketplace

## Project Description  
NexusAPI is a fully decentralized, permissionless API marketplace built on the Stellar network using Soroban smart contracts. It bridges the gap between data providers and consumers by eliminating centralized middlemen, platform fees, and gatekeepers. Developers can monetize their endpoints directly, while consumers can access reliable data streams through secure, trustless crypto payments.

## What it does  
The platform acts as an automated escrow and registry for API services:
1. **Providers** can list their API endpoints and set a monthly subscription price in crypto (e.g., USDC or XLM).
2. **Consumers** can browse available APIs and subscribe by locking the required funds directly into the smart contract.
3. **The Smart Contract** securely holds the funds in escrow, ensuring consumers get the access they paid for while allowing providers to withdraw their hard-earned revenue trustlessly.

## Features  
* **100% Permissionless Architecture:** Anyone can list an API or subscribe to one. No admins, no KYC, no centralized approval process.
* **Trustless Escrow:** Payments are handled entirely on-chain. Funds are locked in the contract and only releasable to the verified provider of that specific API.
* **Direct Monetization:** Providers keep their revenue without losing percentages to traditional Web2 platform fees.
* **Soroban Powered:** Built using Stellar's fast, low-cost, and developer-friendly Rust smart contract environment.
